# react_streams

> A REST-Based video streaming React App with Google Auth


## Quick Start

```bash
# Install dependencies
cd api, client, rtmpserver -> npm install

# Serve client on localhost:3000
npm start

# Serve api
cd api -> npm start

# Serve rtmpserver
cd rtmserver -> npm start

```
